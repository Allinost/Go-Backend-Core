# API 文档索引

---

## 通用约定

### 基础路径

```
http://localhost:29090/api/v1/{module}/
```

### 统一响应格式

```json
// 成功
{"code": 0, "message": "success", "data": {...}}

// 失败
{"code": 1001, "message": "请求参数错误", "detail": "字段校验失败(仅开发环境)"}
```

### 错误码

| code | message | HTTP 状态 |
|------|---------|-----------|
| 0 | success | 200 |
| -1 | 系统繁忙 | 500 |
| 1000 | 系统内部错误 | 500 |
| 1001 | 请求参数错误 | 400 |
| 1002 | 资源不存在 | 404 |
| 1003 | 未授权 | 401 |
| 1004 | 权限不足 | 403 |
| 1005 | 请求频率超限 | 429 |
