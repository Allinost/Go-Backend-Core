# P17 — 公共服务基础设施

---

## 新增服务

| # | 服务 | 文件 | 测试 | 说明 |
|---|------|------|------|------|
| 1 | **feature** — 特性开关 | 2 | 8 | 内存 Flag 管理，支持注册/启用/禁用/列表/查询 |
| 2 | **template** — 模板引擎 | 2 | 8 | 基于 Go `text/template` + `html/template`，fs.FS 驱动，内置 lower/upper/title/default 函数 |
| 3 | **ws** — WebSocket/SSE 实时推送 | 2 | 12 | Hub 模式：客户端注册/房间管理/广播/SSE Writer |
| 4 | **storage** — 文件存储抽象 | 2 | 11 | Storage 接口：Upload/Download/Delete/List/Exists/SignedURL |
| 5 | **search** — 全文搜索 | 2 | 10 | Bleve 引擎：索引/搜索/批量/删除/标签过滤/高亮 |
| 6 | **audit** — 审计追踪 | 2 | 9 | Store 接口 + Service + 辅助函数 RecordLogin/Logout/Create/Update/Delete/Failure |
| 7 | **ai** — AI 推理 | 4 | 11 | Manager + OllamaProvider + OpenAIProvider，支持路由/fallback/流式 |
| 8 | **notification** — 通知服务 | 2 | 10 | Sender 接口 + 多 Sender 回退 + LogSender 桩 |
| 9 | **tracing** — 分布式追踪 | 2 | 9 | Tracer 接口 + noopTracer + logTracer (结构化日志) |
| 10 | **discovery** — 服务发现 | 2 | 11 | Registry 接口 + 内存注册表 + Watch 机制 |

## 已知妥协项

### P1 — 必须发布前解决

| 妥协项 | 说明 |
|--------|------|
| **search 标签缺失** | 已修复：SearchResult 现在返回 tags 字段 |
| **tracing nowMs 硬编码为 0** | 已修复：改为 `time.Now().UnixMilli()` |
| **tracing generateID 为自增计数器** | 已修复：改为 `crypto/rand` 8 字节 hex |
| **template readFile filepath 模式不可用** | 已修复：明确只支持 fs.FS |
| **template 死代码** | 已修复：移除废弃 filepath 逻辑 |
| **ws hub Client.Rooms nil map panic** | 已修复：JoinRoom 时自动初始化 |
| **ai ai.go 孤儿 io.Reader 检查** | 已修复：替换为 `io.Discard` |

### P2 — 当前为内存/桩实现，需生产化

| 妥协项 | 优先级 | 说明 | 状态 |
|--------|--------|------|------|
| feature 无持久化后端 | P2 | 建议接入 Redis 或 database 持久化 | ✅ MySQLStore + SyncManager |
| template 不嵌入模板文件 | P2 | 适合用 `//go:embed` 嵌入到 binary | ✅ 重构为 fs.FS 驱动 |
| ws 缺少 WebSocket Upgrade handler | P2 | 需用 `gorilla/websocket.Upgrader` 实现真正的 WebSocket 连接 | ✅ WSHandler 实现 |
| ws 无 SSE HTTP handler | P2 | 需实现 `gin.Context.Stream()` SSE 端点 | ✅ SSEHandler 实现 |
| storage 仅 MemoryStore | P2 | 需实现 S3/MinIO 或本地文件系统 Store | ✅ S3Store (minio-go) |
| audit 仅 MemoryStore | P2 | 需实现 database-backed Store（MySQL/PostgreSQL） | ✅ MySQLStore 实现 |
| notification 仅 LogSender | P2 | 需实现真实 Email/SMS/Push Sender | ✅ EmailSender (SMTP) |
| tracing 仅 logTracer | P2 | 需集成 OpenTelemetry (Jaeger/Tempo) | ✅ OpenTelemetryTracer 包装 |
| discovery 仅 MemoryRegistry | P2 | 需集成 Consul/etcd | ✅ EtcdRegistry 包装 |
| discovery TTL 未强制过期 | P2 | 需后台 goroutine 定期检查并标记下线 | 待实现 |

| 妥协项 | 说明 | 状态 |

| ws Client.Rooms 由调用方初始化 | 已修复，JoinRoom 自动初始化 | ✅ |
| ws BroadcastToRoom 非原子快照 | 高并发下可能漏送晚加入客户端 | 待处理 |
