## pkg/net/ — 核心客户端

### HTTP 客户端

```go
cfg := net.DefaultHTTPConfig()
cfg.Timeout = 10 * time.Second
cfg.RetryMax = 3
client := net.NewHTTPClient(cfg)

// GET/POST/PUT/DELETE
resp, _ := client.Get(ctx, "https://api.example.com/data")
resp, _ := client.Post(ctx, url, body, "application/json")

// 自定义方法和请求头
resp, _ := client.Do(ctx, "PATCH", url, body, map[string]string{
    "Authorization": "Bearer xxx",
})
```

### TCP 客户端

```go
conn, _ := net.DialTCP(ctx, "nas.local:3493", net.DefaultTCPConfig())
resp, _ := conn.Send([]byte("GET VAR ups.status\n"))
conn.Close()
```

### DNS 解析

```go
result, _ := net.DefaultResolveHost(ctx, "example.com")
// result.Addresses = ["93.184.216.34"]
```

### 重试

```go
err := net.Retry(ctx, net.DefaultRetryConfig(), func(ctx context.Context) error {
    return someOperation(ctx)
})
```

## 配置

```yaml
net:
  http:
    default_timeout: 30s
    max_retries: 3
    retry_wait_min: 500ms
    retry_wait_max: 5s
    max_idle_conns: 100
    idle_conn_timeout: 90s
    proxy:                              # 全局代理（可选）
      http:  "http://proxy:8080"
      https: "http://proxy:8080"

  tcp:
    connect_timeout: 5s
    read_timeout: 10s
```
