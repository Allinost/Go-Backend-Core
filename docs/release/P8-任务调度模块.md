## 当前架构

```
scheduler 触发 (robfig/cron)
  │
  ├─ cron / interval 事件 → asynq.Client.Enqueue()
  │                           │
  │                           └─ asynq.Server.ProcessTask()
  │                                ├─ store.CreateTaskLog(status=running)
  │                                ├─ handler.Execute(ctx, payload)
  │                                └─ store.UpdateTaskLog(status=success/failed)
  │
  ├─ RunTaskNow API → asynq.Client.Enqueue() 或直接 goroutine（asynq 不可用时）
  │
  └─ Start() → store.ListActiveTasks() → 恢复调度
```

## 测试覆盖

- `Store` 接口抽象：`InMemoryStore`（单元测试用）/ `MySQLStore`（生产）
- 模块不依赖 MySQL 即可测试 → `go test` 无需外部服务
- asynq 需 Redis 连接，`module.go` 中 `Init()` 回退不启动
