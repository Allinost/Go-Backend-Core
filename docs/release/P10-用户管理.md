# P10 — 用户管理模块

---

## 已实现的功能

| 功能 | 状态 | 说明 |
|------|------|------|
| User 模型扩展 | ✅ | Email / AvatarURL / Phone / Status(active/inactive/suspended) |
| 软删除 | ✅ | GORM `DeletedAt` + InMemory `DeletedAt` 过滤 |
| 更新个人资料 | ✅ | `PUT /api/v1/auth/profile` — 🔒 修改 nickname/email/avatar/phone |
| 修改密码 | ✅ | `PUT /api/v1/auth/password` — 🔒 需要旧密码验证 |
| 用户列表 | ✅ | `GET /api/v1/auth/users` — 🔒 admin, 分页 + search |
| 用户详情 | ✅ | `GET /api/v1/auth/users/:id` — 🔒 admin, 含 roles |
| 更新用户 | ✅ | `PUT /api/v1/auth/users/:id` — 🔒 admin |
| 删除用户 | ✅ | `DELETE /api/v1/auth/users/:id` — 🔒 admin, 软删除 |
| UserStore 接口扩展 | ✅ | `ListUsers` / `UpdateUser` / `DeleteUser` / `ChangePassword` |
| 状态检查 | ✅ | inactive/suspended 用户拒绝登录 |
| 测试覆盖 | ✅ | auth 服务(35) + 模块(22) + 中间件(7) = **64 个测试** |

## 路由总览

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| PUT | `/api/v1/auth/profile` | 🔒 本人 | 修改昵称/邮箱/头像/手机号 |
| PUT | `/api/v1/auth/password` | 🔒 本人 | 修改密码 |
| GET | `/api/v1/auth/users` | 🔒 `user:admin` | 用户列表(分页+搜索) |
| GET | `/api/v1/auth/users/:id` | 🔒 `user:admin` | 用户详情 |
| PUT | `/api/v1/auth/users/:id` | 🔒 `user:admin` | 更新用户 |
| DELETE | `/api/v1/auth/users/:id` | 🔒 `user:admin` | 删除用户(软删除) |
